# 🏫 School Website

A modern school website built with **React** and **Vite**, featuring smooth animations powered by Framer Motion.

## 🚀 Live Demo

Visit the live site: [https://your-username.github.io/school-website/](https://your-username.github.io/school-website/)

## 🛠️ Tech Stack

- ⚛️ **React 19** — UI library
- ⚡ **Vite** — Fast build tool
- 🎞️ **Framer Motion** — Animations
- 🔗 **React Router DOM** — Client-side routing
- 🎨 **React Icons** — Icon library

## 📦 Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/school-website.git

# Navigate to project directory
cd school-website

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

## 🌐 Deployment
